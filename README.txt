Django-Axilent
--------------

Django-Axilent is plugin for Django that allows easy integration into the Axilent platform.

Django-Axilent will accomplish the following goals:

1.  Storage of an Axilent profile in association with an auth_user model, or within a session.
2.  Mapping between views and Triggers.
3.  Mapping between signals and Triggers.
4.  Mapping between models and Axilent content - post updates on save signal.
5.  Programmatic Content Channel or Content Channel Group access.

Django-Axilent will rely on Sharrock for RPC services to Axilent.
